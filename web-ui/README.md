# FlowForge Web UI

React/TypeScript dashboard for managing FlowForge services.

## Features

- Service catalog and status monitoring
- Interactive API playground
- Usage analytics dashboard
- API key management

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- TanStack Query
- React Router

## Quick Start

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
npm run preview
```
