# FlowForge Zapier Integration

Connect FlowForge to thousands of apps via Zapier.

## Setup

1. Search for "FlowForge" in Zapier
2. Connect your FlowForge account
3. Start building Zaps!

## Available Actions

- Hash Data
- Encrypt Data
- Decrypt Data
- Calculate Expression
- Generate PDF
- Extract Text (OCR)
- Process Image
- AI Chat Completion
- Vector Search
- Transform Data

## Triggers

- Webhook (receive data for processing)
