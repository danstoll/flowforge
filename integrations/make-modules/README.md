# FlowForge Make Modules

Custom modules for Make (formerly Integromat).

## Installation

Import the custom app from the Make marketplace or use the custom app import feature.

## Available Modules

- **Hash Data** - Create hashes
- **Encrypt/Decrypt** - Cryptographic operations
- **Calculate** - Mathematical expressions
- **Process PDF** - PDF operations
- **Extract Text (OCR)** - Image to text
- **Process Image** - Image manipulation
- **AI Completion** - LLM inference
- **Vector Search** - Similarity search
- **Transform Data** - Format conversions
